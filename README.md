## 使用推荐

### css书写

* 脚手架默认情况下未开通css module功能，实际上个人也推荐使用BEM的书写方式
* 全局变量使用的css自定义属性，详情请参考[使用CSS自定义属性（变量）](https://developer.mozilla.org/zh-CN/docs/Web/CSS/Using_CSS_custom_properties)
